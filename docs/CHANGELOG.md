# 更新日志

本项目版本记录 notable 变更。格式借鉴 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)。

## [0.5.2] - 2026-07-24

git 同步、MCP 接入、速记打磨、七语言国际化，以及自动更新与发布链路完善。

### git 同步（技术用户的免费同步方案）

- vault 接入任意 git 远端（GitHub / cnb 等）：设置抽屉填地址 + 用户名 + 令牌即用
- 变更 3 分钟无操作自动提交；定时推拉（默认 10 分钟，1~1440 分钟可配，即改即存）；「立即同步」手动触发
- diff3 三方合并：不同段落自动合并，同行冲突写入标准 conflict markers；二进制附件冲突保留双方
- 便签外部变更感知：未编辑自动重载，已编辑横幅二选一；冲突便签列表标识 + 设置计数
- 冲突解决视图：markers 内容切换等宽原文编辑，「全部用本地/远端」一键剥离，保存解决后自动触发同步

### MCP 服务（AI 助手读写便签）

- pinslipd 内嵌 MCP server（Streamable HTTP，仅监听 127.0.0.1，随应用启停）
- 11 个工具：搜索/列表/读取/新建/替换/追加/删除（只进回收区）/标签/文件夹 + 同步触发/状态查询
- `.pinslip/mcp.json` 发现文件（端口/pid/版本）；设置抽屉 MCP 小节：开关、接入信息、一键复制客户端配置
- 附 `skills/pinslip/SKILL.md` 接入指南（Kimi Work / Claude Code 配置示例）

### 速记打磨

- 关窗不丢内容：失焦/Esc 有内容自动保存再关
- Ctrl+Enter 保存并继续连记；剪贴板文本开窗带入（可关）；Ctrl+V 贴图带缩略图条
- 落点可选：逐条新建 / 聚合到「速记 今日」（`### HH:mm` 时间戳追加）
- 窗口跟随鼠标所在屏幕（多屏友好）

### 国际化

- 界面七语言：简中/英/日/韩/西/德/法；默认跟随系统，设置里手动切换即时生效，全部窗口同步跟随
- 覆盖主界面/便签窗/速记窗/托盘/更新提示；服务端错误按 code 映射翻译（原文不变，MCP/日志不受影响）
- 语言包 key 对齐自检脚本（check-i18n.mjs，214 key × 7 语言）

### 自动更新与发布链路

- 自动更新：接入 electron-updater（GitHub Releases），启动 15s 静默检查；设置抽屉新增「更新」区块——当前版本号、手动检查更新、下载进度展示、下载完成后一键重启安装；国内网络自动回退腾讯云 COS 镜像
- 发布链路：electron-builder 配置 publish=github，CI 产物随包携带 latest.yml / latest-mac.yml / blockmap（自动更新元信息）
- mac 出包改单腿双架构：pinslipd 交叉编译后经 lipo 合成 universal 二进制，一次产出 arm64 + x64 双 dmg 并共享同一份 latest-mac.yml（避免双腿互相覆盖更新元信息）

### 修正

- 空态提示排版修正（主列表/文件夹空态居中）
- 设置值列 RTL 倒序仅限路径/URL（修复「上次同步」时间显示为「分钟前 8」）；按钮行长文案换行防撑爆
- 同步配置表单编辑中不被 30s 轮询回填覆盖
- 组手柄提示居中对齐 + 多行折行（长文案不溢出）
- 复制全部过滤 `<br>` 空行标记

## [0.5.0] - 2026-07-22

首个定版发布（Phase 1 收官）。从便签骨架到完整桌面便签 + 本地知识管理工具。

### 便签本体

- 无边框透明桌面便签，多窗口并存，全部进任务栏便于找回
- 置顶可选（红色图钉，呼应 logo）；不置顶也可被磁铁/边缘吸附找回
- 六种便签色（黄/粉/绿/蓝/紫/橙），右侧弹出颜色条 + 当前色图钉标记
- 折叠成标题条（180ms 卷起动画，展开尺寸记忆；脏数据自动回退默认高度）
- 边缘与右下角拖拽缩放（最小 220×160）；focus 时浮现底部工具栏
- 底部工具栏按优先级自适应宽度：加粗/删除线/任务列表/添加图像 ｜ 标签/文件夹/保存状态/复制全部/更多

### 编辑器

- Milkdown 所见即所得 Markdown 编辑（标题/列表/任务/引用/代码等快捷键）
- 任务列表可点击 checkbox（自定义 nodeview）
- 粘贴图片自动存附件（相对笔记路径，Obsidian 直接可读）
- 长文滚轮/方向键滚动，滚动条自动隐藏

### 存储与兼容（硬约束：与 Obsidian 完全互通）

- 用户自选 vault 目录（Obsidian 式），便签 = YAML frontmatter + Markdown 的 `.md` 文件
- 文件名 `标题-日期-id.md`，标题随首行内容自动推导并重命名
- 外部修改/删除文件实时监听同步；幽灵窗口自动关闭
- 图片写相对路径，外部编辑器打开不裂图

### 吸附与便签组

- 屏幕边缘吸附（98% 视口阈值内全屏任意边），落位动画跟手
- 便签间磁铁：置顶便签水平/垂直靠近自动对齐贴边（重叠 ≥50% 不触发磁铁）
- **便签组**：重叠 50% 松手成组——左对齐竖向叠放、2px 间距、拼缝外框；
  组手柄（顶边药丸）拖动整组/双击命名/右键解散；成员折叠展开兄弟自动贴上让位；
  组宽统一（最宽者为准）；关窗即退组；会话恢复自动归位 + 出屏钳制

### 主界面（管理入口）

- 三视图：列表（全部平铺，按更新/创建时间升降序）/ 文件夹（分层导航+面包屑）/ 标签（频次分组，可折叠）
- 全文搜索：中文 bigram + bm25 加权（标题 10 > 标签 5 > 正文），命中词高亮 + 摘要窗口锚定
- 文件夹嵌套：新建/行内重命名/删除（便签上移根目录 或 整树进回收区）
- 移动便签到文件夹（路径筛选框快速定位）
- 设置抽屉：存储位置更改、开机自启、回收区策略

### 回收区

- 删除便签/文件夹进 `.trash`（不真删），保留天数 7/30/90/永不 可配
- 打开回收区手动找回；一键清空（两阶段确认）

### 系统集成

- 托盘常驻（左键主界面/右键菜单）
- 全局快捷键速记浮窗（inbox 暂存区）
- 开机自启（安装版）
- 会话恢复：便签位置/尺寸/折叠态/组关系全部还原；混合 DPI 自校准防漂移；
  出屏窗口自动钳回工作区
