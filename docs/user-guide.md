# PinSlip 用户手册（v0.5）

> PinSlip 是贴在 Windows 桌面上的便签工具：随手钉一张，写了就走。
> 所有便签都是纯 Markdown 文件，存在你自己选的文件夹里，和 Obsidian 完全互通。
> 本手册按「第一次用 → 日常用 → 整理 → 进阶」的顺序讲。

---

## 一、第一次启动

1. 启动后会先让你**选择便签存放位置**（建议 `文档\PinSlip`）。所有便签都以
   Markdown 文件保存在这里，以后可以在「设置」里随时更换。
2. 选完后出现**主界面**（窄条面板，固定在屏幕右下角）和托盘图标。
3. 数据目录结构（可以用任何编辑器打开）：

```
你选的目录/
├── notes/        # 全部便签（标题-日期-id.md）
│   └── 子文件夹/   # 你自己建的分类目录
├── inbox/        # 速记暂存
├── attachments/  # 粘贴的图片
└── .trash/       # 回收区（删除的东西先进这里）
```

## 二、便签的日常操作

### 新建便签

四种方式，随手选：

- 便签标题栏的 **＋**（便签在子文件夹里时会问：建到同文件夹还是根目录）
- 主界面顶部「**新建便签**」
- 主界面文件夹视图里当前目录右侧的 **＋**（直接建到该目录）
- 全局快捷键 **`Ctrl+Shift+N`** 呼出**速记浮窗**（写完进 inbox 暂存区）

### 标题栏（从左到右）

| 按钮 | 作用 |
|---|---|
| 📌 红色图钉 | 置顶 / 取消置顶。置顶的便签永远浮在最上面 |
| 标题 | 自动取正文第一行（markdown 符号会被剥掉），不用手动起 |
| ＋ | 新建便签 |
| 🎨 颜色 | 从右向左弹出颜色条，六色可选；当前颜色上有红色图钉标记 |
| ▁ / ▽ | 折叠成标题条 / 展开（有 180ms 卷起动画，展开后恢复原来高度） |
| × | 关闭便签（内容已保存，组内便签关闭即退组） |

### 底部工具栏（便签获得焦点时浮现）

- **编辑区**：加粗 / 删除线 / 任务列表 / 添加图片
- **信息区**：标签 🏷 / 文件夹 📁 / 保存状态 / 复制全部 / ⋯ 更多（便签列表、删除）
- 窗口太窄时低优先级按钮自动收起，拉宽自动放回

### 编辑器

- 支持 Markdown 快捷键：`#` 标题、`-` 无序列表、`1.` 有序列表、`- [ ]` 任务框、
  `>` 引用、`Ctrl+B` 加粗等
- 任务框可以**直接点击打勾**
- **粘贴图片**：自动存到 `attachments/`，文件里写相对路径（Obsidian 打开不裂图）
- 长文自动滚动，滚动条自动隐藏

### 移动与缩放

- 拖标题栏空白处移动；拖边缘或右下角缩放（最小 220×160）
- **屏幕边缘吸附**：拖到屏幕任意边缘附近自动贴齐
- **便签磁铁**：两张置顶便签靠近时自动对齐贴边（间隔 2px）

## 三、便签组：一叠便签

> 把相关的便签摞成一叠，一起拖、一起收。

- **成组**：拖动一张置顶便签，和另一张重叠超过一半 → 两张同时亮起，
  **松手即成组**。成员左对齐竖向叠放，间距 2px，外圈有拼缝外框
- **加入组**：把自由便签拖到组里任意成员上方重叠一半松手，
  插到被压那张的下面
- **组手柄**：组顶边中点的蓝色药丸
  - **按住拖动** = 整组移动（松手整组做屏幕边缘吸附）
  - **双击** = 给组起名字（Enter 确认，Esc 取消）
  - **右键** = 解散此组（成员原地散开）
- **联动**：折叠一张，下面的兄弟自动上贴；展开自动让位
- **组宽统一**：全组看齐最宽的成员（调窄会被拉回，内容不被挤）
- **退组**：拖出组外 40px 松手；取消置顶；或直接关闭便签
- 重开应用，组和每张便签的位置原样还原

## 四、整理：标签与文件夹

### 标签

- 便签底部工具栏 🏷 → 弹出标签面板：输入回车添加（可点 ＋ 按钮）、点 × 移除
- 有标签时按钮变成实心并显示数量
- 主界面「**标签**」视图：按标签分组展示，频次高的在前；
  **点组标题可以收起/展开**；无标签的便签归在最后一组

### 文件夹

- 便签底部工具栏 📁 → 面板显示当前位置，可**打开目录**、移动到别的文件夹
  （目录多时上方有「搜索目录」筛选框）
- 主界面「**文件夹**」视图：分层浏览，面包屑导航（层级深时中间折叠成 …）
- 当前目录右侧：＋ 新建便签到此目录、📁＋ 新建子文件夹
- 文件夹**重命名**（铅笔图标，行内编辑）；**删除**（垃圾桶图标）：
  空文件夹直接删；非空可选「便签移到根目录」或「整树进回收区」
- 移动便签：列表项的 📂 按钮 → 弹层选目标（带筛选框）

### 搜索

- 主界面顶部搜索框，输入即搜（防抖）
- 中文按 bigram 匹配，权重：**标题 > 标签 > 正文**
- 命中词高亮，长内容自动截取命中位置附近

## 五、回收区

- 删除的便签和文件夹都进 `.trash`，**不会真删**
- 设置 → 回收区：**自动清理天数**（7 / 30 / 90 天 / 永不）
- 「打开回收区」可以把误删的文件拖回 `notes/` 找回（应用自动识别）
- 「清空回收区」需点两次确认

---

## 六、设置（主界面右上角 ⚙）

| 项 | 说明 |
|---|---|
| 开机自启 | 安装版可用，开发模式下置灰 |
| 存储位置 | 显示当前 vault 路径；「打开便签目录」/「更改位置…」（切换后所有窗口重载） |
| 回收区 | 存量统计、自动清理天数、打开/清空 |

---

## 七、系统集成

- **托盘**：左键 = 显示/隐藏主界面；右键 = 菜单
- **速记**：`Ctrl+Shift+N` 全局呼出浮窗，写完即存 inbox，主界面列表顶部可见
- **会话恢复**：退出再开，便签的位置、尺寸、折叠状态、分组全部还原；
  换显示器/拔外接屏后，跑出屏幕的窗口会自动拉回可见区
- **任务栏**：每张便签都有任务栏图标，不置顶也能随时找回

---

## 八、给你的数据上个保险

- 便签是纯文本 Markdown，**任何备份工具都能备份**（直接复制整个 vault 目录）
- 和 Obsidian 共用一个目录也完全没问题：两边都能改，互相实时可见
- 唯一的建议：别在多台机器上同时通过同步盘改同一张便签（后写覆盖先写）
